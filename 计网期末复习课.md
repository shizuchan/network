# 计网期末复习课

[TOC]

1. 理论基础：电路交换 分组交换
2. 网络评价指标：速率  带宽  吞吐量  时延  时延×带宽
3. 体系结构：模块化  -> 两层结构 -> 三层结构
4. 节点互联：编码  ->  组帧  ->  差错  ->  可靠传输  ->  介质访问
5. 差错检测：各个层次共同作用的结果。多层协同。链路层CRC/网络层checksum（校验和，逐跳，校验少）/传输层checksum（端到端，校验多）/
6. 可靠传输：丢包、乱序    ARQ（自动请求重传） （停等 （非流水线）  +滑动窗口（流水线））
7. 网络模式：直连 -> 交换 -> IP网
8. 数据流动：横向 （链路层/邻居，网络层/数据节点，传输层/数据节点远程） +  纵向（Data，Segment，Packet，Frame，bits flow）
9. 路由协议：三种  信息内容+传播范围
10. 分组转发：决策 （找IP，直接/间接）+ 转发（找MAC）
11. 缓存机理  代理：历史信息的再利用  **IPv6集大成应用**
12. 传输控制：端到端，对端控制（流量控制，**显式反馈**） +网络控制（**隐式反馈**）慢启动/拥塞避免/快启动/快重传
13. 网络应用：基础模型  **服务部署**， **本地代理**
14. 网络安全：非对称密码 -->  机密性  签名
